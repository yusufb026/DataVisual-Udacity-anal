Introduction¶

I have decided to chosen the US Census and Gun Dataset to perform my analysis on. I started by unzipping the provided zip file, converting the xlsx files to csv and then uploaded them to the workspace. The goal of this project is to find coorelation between the census data and the gun data.

My Hypothesis - There have to be certain factors which drive people towards wanting guns, and the census data must have some important variables which we can use to determine this. Due to the unclear and complicated nature of the dataset it is difficult to analyse all of the dependent variables and independent ones, however give below are some of the questions which I am trying to answer

  • Is there any coorelation between population and gun ownership?
  • Which are the top 5 states with the highest amount of guns per capita?
  • Which are the top 5 states with the lowest amount of guns per capita?
  • Is there a coorelation between number of veterans and gun ownership?
  • Does poverty and gun ownership have a coorelation?

Data Wrangling¶

Here, we have imported the required datasets. Next step would be to combine them into a single one to make analysis easier.

In the Gun Data - each row represents gun data for those specific months - I decided to combine this all into yearly basis to make it better correspond with the Census data which is year-ed.

import pandas as pd
import numpy as np
from scipy import stats

%matplotlib inline

census_df = pd.read_csv('census_data.csv')
census_pop = dict(census_df.iloc[3][2:])
census_poverty = dict(census_df.iloc[49][2:])
census_veterans = dict(census_df.iloc[20][2:])


# Convert yyyy/mm to yyyy
dateparse = lambda dates: [pd.datetime.strptime(d, '%Y-%m') for d in dates]
gun_df = pd.read_csv('gun_data.csv', parse_dates=['month'], date_parser=dateparse)
gun_df['year'] = gun_df['month'].dt.year
gun_df_year = gun_df.groupby(['year','state']).sum()

# Create a combined dataset using the census and gun datasets
gun_by_state = gun_df_year.query('year == 2016').groupby(['state'])
for idx, row in gun_by_state:
    if idx in census_pop :
        gun_df_year.at[row.index,'population'] = census_pop[idx]
    if idx in census_poverty :
        gun_df_year.at[row.index,'poverty'] = census_poverty[idx]
    if idx in census_veterans :
        gun_df_year.at[row.index,'veterans'] = census_veterans[idx]    


temp_df = gun_df_year.query('year == 2016')

Data Cleaning¶

Now, that we have carefully combined the 2 datasets we can move aggregating the different columns to make things easier to analyse.

We remove all of the % and ',' symbols from the data, we drop the columns which we don't need and convert everything into usable datatypes.

We also take things a step further and delete all of the outliers from the data, after creating a copy of it with the outliers for future use.

Research Question 1 - Is there any coorelation between population and gun ownership?¶

Population vs Gun/capita¶

To compare the relation of guns to population we first calculate the guns to person ratio of the states, then we plot that and the population in a scatter plot.

Answer¶

There is a weak corelation between the population and the number of guns.

It is apparent from the visualizations that scarely populated states seem to have more number of guns, and larger cities seem to have relatively lesser number of guns.

My reasoning for this is - in smaller states the settlements are usually away from each other, people might think that the response times for law enforcement are extremely high. To counter this they might believe that owning a gun would mean they could protect themselves and their families better. In more populated cities the law enforcement is able to respond more quickly hence people might feel less of a need for weapons.

Conclusions¶

We can now summarise all our observations.

Gun ownership seems to be strongly driven more by emotion rather than any specific environmental factors. That being said, it does not mean that there are not quantifiable and measureable statistics for it.

Population and people who have had experience with guns before seem to be some of the driving forces for gun ownership. The Crime rate could also be an important factor affecting ownership but the linked dataset did not have any crime rate values.
